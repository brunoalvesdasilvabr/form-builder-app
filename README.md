# Form Builder (Angular 19)

Drag-and-drop form builder with a **component palette** (Input, Checkbox, Radio, Table) and a **canvas table** where you can drop components into cells, add rows/columns, and merge cells.

## Features

- **Palette**: Draggable components â€” Input, Checkbox, Radio, Table
- **Canvas**: Table with droppable cells; drag a component from the palette and drop it into any cell
- **Add row / Add column**: Toolbar buttons to grow the canvas
- **Merge cells**: Click one cell, then click another to select a range; click **Merge selected** to merge. Right-click a merged cell to unmerge

## Run the app

```bash
cd form-builder-app
npm install
npm start
```

Open http://localhost:4200

## Build

```bash
npm run build
```

Output is in `dist/form-builder-app`.
